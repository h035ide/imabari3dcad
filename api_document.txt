/**
 * スケッチ平面を作成します。
 * @function CreateSketchPlane
 * @param {文字列} ElementName - 作成するスケッチ平面の名称です。[空文字可]
 * @param {要素グループ} ElementGroup - 所属する要素グループです。[空文字可]
 * @param {平面} PlaneDef - スケッチ平面を作成する平面を指定します。
 * @param {長さ} PlaneOffset - 平面からのオフセット距離を指定します。
 * @param {bool} bRevPlane - スケッチ平面を反転するかのフラグです。
 * @param {bool} bRevUV - スケッチ平面のX,Y軸を交換するかのフラグです。
 * @param {注記スタイル} StyleName - 適用する注記スタイル名称です。[空文字可]
 * @param {点} OriginPoint - スケッチ平面の原点を指定します。[空文字可]
 * @param {方向} AxisDirection - スケッチ平面の軸方向を指定します。[空文字可]
 * @param {bool} bDefAxisIsX - スケッチ平面のX軸を指定する場合はTrueです。
 * @param {bool} bUpdate - 更新フラグ（未実装）です。
 * @returns {要素ID} 作成されたスケッチ平面要素のID。
 */
