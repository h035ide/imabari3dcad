取付部材設定

板部材に取り付ける部材を設定します。部品図に出力されるときに取りつく部材名がマーキン線とともに出力されます。

設定されない場合、マーキン面側についている部材はマーキン線のみ出力します。マーキン面と反対側の面側の部材は出力しません。

このコマンドは部材が正しくとりつく位置あるかどうかのチェックは行いません。

パラメータ

操作方法

  • マーキン面の指定

    指定する部材を「ベース板」として選択します。現在のマーキン面の状態で「モールド面をマーキン面とする」のチェックが更新されます。

    マーキン面のデフォルトはモールド面となっています。反対側をマーキン面とするために「モールド面をマーキン面とする」のチェックをはずして、ＯＫボタンで確定します。

  • 取付部材の設定

    ベース板、取付部材をそれぞれ選択して、ＯＫボタンで確定します。

  • 取付除外部材の設定

    ここで指定される部材はベース板にマーキン線等を出力しなくなります。

    ベース板、取付除外部材をそれぞれ選択して、ＯＫボタンで確定します。

  • 取付部材の解除

    ベース板を選択して、「取付部材の解除」ボタンで設定されている全ての取付部材が解除されます。

  • 取付除外部材の解除

    ベース板を選択して、「取付除外部材の解除」ボタンで設定されている全ての取付除外部材が解除されます。

  • 取付部材の確認

    ベース板を選択して「設定部材の確認」ボタンで設定されている部材が「取付部材」として選択されます。

  • ベース板の確認

    ベース板を確認したい部材１つを「取付部材」として選択します。「ベース板の確認」ボタンで設定されている場合はベースの部材が「ベース板」として選択されます。

  • 特定の取付部材の解除

    解除したい部材を「取付部材」として選択します。「設定を解除」ボタンで取付部材設定が解除されます。
