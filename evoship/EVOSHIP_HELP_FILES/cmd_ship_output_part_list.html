部材データ出力

モデル内の部材の各種データをCSVフォーマットでファイル出力します。

操作方法

ファイル選択ダイアログが表示されますので、出力ファイル名を指定します。

ファイルのフォーマット

csvの形式で、一行が一つの変数定義となります。
各行は左から以下の内容のカラムになります。

  • ブロック名

    部材のあるブロック名称

  • 部材名

    部材名称

  • 部材種別

    部材種別毎の文字列です。文字列は以下の内容になります。

    板材

    • COL (カラープレー)
    • BKT (ブラケット)
    • DBL (ダブリング)
    • PL (その他板材)

    条材

    • FB (平鋼)
    • AB (等辺山形鋼)
    • ABS (不等辺山形鋼)
    • NAB (不等辺不等厚山形鋼)
    • BP (球平形鋼)
    • I (I形鋼)
    • H (H形鋼)
    • T (T形鋼)
    • CB (溝形鋼)

  • 部材寸法

    部材種別が板材の場合は板厚(mm),条材の場合は断面の形状寸法(mm)です。

  • 曲げ加工

    板材や条材を曲げ加工する必要がある場合にはB,そうでない場合は空欄です。

  • 材質

    部材の材料名称

  • 表面積

    部材の表面積(mm2)

  • 体積

    部材の体積(mm3)

  • 重量

    部材の重量(Kg)

  • 重心

    部材の重心位置(mm)のX座標値,Y座標値,Z座標値

  • 慣性モーメント

    部材重心の慣性モーメント(kg X mm2),Lxx,Lxy,Lxz,Lyx,Lyy,Lyz,Lzx,Lzy,Lzz

  • 慣性主軸

    部材の慣性主軸の方向ベクトル成分X,Y,Z。Ix(X,Y,Z),Iy(X,Y,Z),Iz(X,Y,Z)

  • 主慣性モーメント

    部材の主慣性モーメント(kg X mm2),Px,Py,Pz
