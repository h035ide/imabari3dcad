スプライン（ファイル）(カーブ）

点列を結んだスプラインカーブを作成します。点列はファイルから読込みます。

点データを選択するダイアログが表示されます。
ShiftキーやCtrlキーを使用して複数のファイルを一度に選択できます。

  • 複数カーブ（複数カーブのフォーマットの場合はチェックします。)
  • 単位

    ファイルの座標値データの単位を指定します。

ファイルフォーマット

データがコンマ、空白あるいは改行で区切られたテキストファイルが利用できます。
データは以下のように通過点の座標値が順に並びます。

  • 複数カーブ

    点の数　 Ｘ座標値 Ｙ座標値 Z座標値　…　…　点の数　…

  • 1本のカーブ

    Ｘ座標値 Ｙ座標値 Z座標値　…
