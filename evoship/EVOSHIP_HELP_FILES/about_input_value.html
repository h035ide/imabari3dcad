数値入力


Evo.Shipの数値入力のためのエディットボックスには以下のような入力補助機能があります。
  • 変数入力機能

    コンテキストメニューから変数を選択（作成）できます。

  • 関数電卓機能

    数学関数(sin,cos など),単位(mm,m,cm,in,inch,ft,feet,rad,deg など)を含んだ計算式が入力できます。(使用できる定数,単位,関数)
    (単位の演算は四則演算に優先します。)
    変数を含む式も入力できます。
    例)

    • 4+3*5
      ->19mm
    • 4mm+5m*2
      ->10004mm
    • (4+5)*2
      ->18mm
    • 50*sin(30)
      ->25mm
    • V1+10
      ->20mm(V1が10mmの時)
    • 400m2/20m
      ->20000mm
    • 20m*20m
      ->エラー(面積となるので）
