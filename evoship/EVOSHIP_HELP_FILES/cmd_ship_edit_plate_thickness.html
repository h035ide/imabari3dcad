板厚変更

モールド位置を基準に部材の板厚を変更します。

操作方法

板厚を変更する部材ソリッドを選択(複数可)します。最初に選択された部材の板厚がエディットボックスに設定されます。
板厚を入力してOKボタンで実行します。

板厚を変更することができるソリッド

Evo.Shipは板形状だけでなく様々な形状を作成することができるので、どんな形状でも板厚を変更できるわけではありません。
このコマンドでは以下のような要素の板厚を変更できます。

  • 以下の船殻機能で作成されたソリッドとその派生要素(元要素に関連をもった各種コピー、分割等で作成されたソリッド)
    • プレート
    • ブラケット
    • ダブリング
  • 以下のフィーチャーで作成されたソリッドとその派生要素(元要素に関連をもった各種コピー、分割等で作成されたソリッド)
    • 押し出し

      厚みづけされて作成された場合はその板厚、そうでない場合は押し出し方向の高さを板厚とします。

    • 回転(厚みづけされて作成された場合)
    • スイープ(厚みづけされて作成された場合)
    • 厚み付け(厚みづけされて作成された場合)
    • 境界指定
